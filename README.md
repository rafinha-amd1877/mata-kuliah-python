# CatatPanen

Program sederhana untuk mencatat, mengelola, dan mengekspor data hasil panen petani.

## Fitur
- Mencatat data hasil panen  
- Melihat seluruh data  
- Menghapus dan mengupdate data  
- Ekspor data ke Excel menggunakan `pandas` dan `openpyxl`

## Teknologi
- Python  
- Pandas  
- OpenPyXL  
- Tabulate

## Persyaratan
- Python 3.x  
- pip

## Instalasi
1. Clone atau download repository ini  
2. Masuk ke folder project  
3. Install dependency:
   ```bash
   pip install pandas openpyxl tabulate
